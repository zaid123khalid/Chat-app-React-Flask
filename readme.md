React & Flask Chat App<br/>

Requirements (backend):<br/>
Flask<br/>
Flask-socketio <br/>
Flask-sqlalchemy <br/>
flask-login <br/>
sqlalchemy <br/>
passlib <br/>
pymysql <br/>

Requirements (frontend):<br/>
react-router-dom<br/>
socket.io-client<br/>
@emotion/react<br/>
@emotion/styled<br/>
@mui/material<br/>

For frontend:<br/>
First run `npm i` or `npm install`<br/>
then run npm i `package-name`<br/>

For backend:<br/>
Install libraries using pip or your favourite package manager
